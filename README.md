## Калькулятор расстояния между координатами

В файле `HomeWorks/JS/4-navigator/app.js` реализован расчет расстояния между двумя точками на плоскости.

### Добавленная функциональность:
- Расчет расстояния по формуле Пифагора: `d = √((x2 - x1)² + (y2 - y1)²)`
- Вывод результата в консоль

### Переменные:
- `addressLat` - координата lat адреса назначения (10)
- `addressLong` - координата long адреса назначения (50) 
- `positionLat` - текущая координата lat пользователя (1)
- `positionLong` - текущая координата long пользователя (15)

### Результат:
Расстояние между текущим положением и адресом назначения выводится в консоль.

## Сортировка массивов

В файле `HomeWorks/JS/9-sort-loops/app.js` реализован алгоритм пузырьковой сортировки.

### Добавленная функциональность:
- Функция `sort` для сортировки массива по возрастанию
- Алгоритм пузырьковой сортировки с двойным циклом
- Обмен элементов через временную переменную
- Чистая реализация без дополнительного логирования

### Исходный массив:
```javascript
const arr = [1, 40, -5, 10, 0];
```

### Алгоритм сортировки:
```javascript
for (let i = 0; i < arr.length; i++) {
  for (let j = 0; j < arr.length; j++) {
    if (arr[j] > arr[j+1]) {
      let el = arr[j];
      arr[j] = arr[j+1];
      arr[j+1] = el;
    }
  }
}
```

### Результат:
Функция сортирует массив по возрастанию методом пузырьковой сортировки и выводит отсортированный массив в консоль.

## Очистка массивов

В файле `HomeWorks/JS/10-array-clean/app.js` реализована функция для фильтрации массивов.

### Добавленная функциональность:
- Функция `arr` для удаления элементов из массива по условию
- Использование метода `filter()` для фильтрации
- Функция высшего порядка, принимающая callback
- Инверсия логики через оператор `!`

### Параметры функции:
- `array` - исходный массив для фильтрации
- `removeFunction` - функция-условие для удаления элементов

### Логика работы:
```javascript
const arr = (array, removeFunction) => {
  return array.filter(item => !removeFunction(item));
};
```

### Пример использования:
```javascript
arr([3, 6, 9, 2], num => num > 5) // возвращает [3, 2]
```

### Результат:
Функция возвращает новый массив без элементов, удовлетворяющих условию функции `removeFunction`.

## Алгоритм Луна для валидации карт

В файле `HomeWorks/JS/12-luna/app.js` реализован алгоритм Луна для проверки корректности номеров банковских карт.

### Добавленная функциональность:
- Удаление разделителей из номера карты
- Умножение каждой второй цифры на 2 (с позиции 0)
- Вычитание 9 из результатов >= 9
- Суммирование всех цифр методом `reduce()`
- Проверка кратности суммы 10

### Тестовый номер карты:
```javascript
let cardNumber = "4561-2612-1234-5467";
```

### Алгоритм обработки:
```javascript
// Удаление дефисов и преобразование в массив
cardNumber = cardNumber.replaceAll("-", "").split('');

// Обработка каждой второй цифры
for (let i = 0; i < cardNumber.length; i += 2) {
  let firstStep = parseInt(cardNumber[i]) * 2;
  if (firstStep >= 9) {
    firstStep = firstStep - 9;
  }
  cardNumber[i] = firstStep.toString();
}

// Суммирование и проверка
const sumNumbers = cardNumber.reduce((acc, value) => acc + Number(value), 0);
const result = sumNumbers % 10 == 0;
```

### Результат:
Алгоритм возвращает `true` для валидных номеров карт и `false` для невалидных по алгоритму Луна.
