- Алгоритм пузырьковой сортировки с двойным циклом
- Обмен элементов через временную переменную
- Чистая реализация без дополнительного логирования

### Исходный массив:
```javascript

```

### Алгоритм сортировки:
```javascript
for (let i = 0; i < arr.length; i++) {
  for (let j = 0; j < arr.length; j++) {
    if (arr[j] > arr[j+1]) {
      let el = arr[j];
      arr[j] = arr[j+1];
      arr[j+1] = el;
    }
  }
}
```

### Результат:
Функция сортирует массив по возрастанию методом пузырьковой сортировки и выводит отсортированный массив в консоль.

## Алгоритм Луна для валидации карт ↑ [Назад к навигации](#навигация)

В файле `HomeWorks/JS/12-luna/app.js` реализована функция `luna` для проверки корректности номеров банковских карт.

### Добавленная функциональность:
- Функция `luna(cardNumber)` с параметром номера карты
- Удаление разделителей из номера карты
- Умножение каждой второй цифры на 2 (с позиции 0)
- Вычитание 9 из результатов >= 9
- Суммирование всех цифр методом `reduce()`
- Возврат результата проверки кратности суммы 10

### Тестовый номер карты:
```javascript
let cardNumber = "4561-2612-1234-5464";
```

### Функция валидации:
```javascript
const luna = (cardNumber) => {
  cardNumber = cardNumber.replaceAll("-", "");
  cardNumber = cardNumber.split('');

  for (let i = 0; i < cardNumber.length; i += 2) {
    let firstStep = parseInt(cardNumber[i]) * 2;
    
    if (firstStep >= 9) {
      firstStep = firstStep - 9;
    }
    
    cardNumber[i] = firstStep.toString();
  }

  const sumNumbers = cardNumber.reduce((acc, value) => {
    return acc = Number(acc) + Number(value);
  })

  return sumNumbers % 10 == 0
}
```

### Результат:
Функция `luna()` возвращает `true` для валидных номеров карт и `false` для невалидных по алгоритму Луна.

## Генерация query-строк ↑ [Назад к навигации](#навигация)

В файле `HomeWorks/JS/13-query/app.js` реализована функция для преобразования объекта в query-строку URL.

### Добавленная функциональность:
- Функция `outputString(queryParams)` для генерации URL параметров
- Использование `Object.entries()` для получения пар ключ-значение
- Применение метода `map()` для трансформации данных
- Объединение параметров символом `&` через `join()`

### Исходный объект параметров:
```javascript
const queryParams = {
  search: 'Вася',
  take: 10
};
```

### Подробное описание функции:
```javascript
const outputString = (queryParams) => {
  return Object.entries(queryParams)    // Преобразует объект в массив пар [key, value]
    .map(([key, value]) => `${key}=${value}`)  // Каждую пару превращает в строку "key=value"
    .join('&');                         // Объединяет все строки символом "&"
};
```

### Пошаговый разбор:
1. **`Object.entries(queryParams)`** - преобразует объект `{search: 'Вася', take: 10}` в массив `[['search', 'Вася'], ['take', 10]]`
2. **`.map(([key, value]) => ...)`** - использует деструктуризацию для извлечения ключа и значения из каждой пары
3. **`${key}=${value}`** - создает строку формата "ключ=значение" для каждого параметра
4. **`.join('&')`** - объединяет все строки в одну, разделяя их символом "&"

### Пример работы:
```javascript
outputString({search: 'Вася', take: 10}) // возвращает "search=Вася&take=10"
```

### Результат:
Функция преобразует JavaScript объект в стандартную query-строку формата `key1=value1&key2=value2`, которую можно использовать в URL для передачи параметров.

- `main.js` - JavaScript логика вычислений

### HTML структура:
```html
<div class="calculator">
  <h1>Simply Clever Calculator</h1>
  <input type="number" id="firstValue" required>
  <select id="selection">
    <option value="+">+</option>
    <option value="-">-</option>
    <option value="*">*</option>
    <option value="/">/</option>
  </select>
  <input type="number" id="secondValue" required>
  <button onclick="resultButton()">||</button>
  <p id="result">0</p>
</div>
```

### JavaScript функциональность:
- Функция `resultButton()` для выполнения вычислений
- Получение значений из полей ввода через `getElementById()`
- Определение операции из выпадающего списка
- Выполнение арифметических операций: сложение, вычитание, умножение, деление
- Валидация результата с проверкой на `NaN` и бесконечность
- Обработка деления на ноль с выводом предупреждения

### Логика вычислений:
```javascript
if (selectValue == '+') {
  result = Number(firstValue) + Number(secondValue);
} else if (selectValue == '-') {
  result = Number(firstValue) - Number(secondValue);
} else if (selectValue == '*') {
  result = Number(firstValue) * Number(secondValue);
} else if (selectValue == '/') {
  result = Number(firstValue) / Number(secondValue);
}
```

### Обработка ошибок:
- **NaN проверка:** если результат не является числом, выводится "Ошибка!"
- **Деление на ноль:** при получении `Infinity` или `-Infinity` выводится "Деление на ноль недопустимо"
- **Преобразование типов:** входные значения принудительно преобразуются в числа через `Number()`

### CSS оформление:
- Адаптивный дизайн с flexbox-контейнером
- Стилизованные поля ввода с border-radius
- Кастомный select без нативной стрелки

### Возможности интерфейса:
- Два поля для ввода чисел (включая отрицательные и дробные)
- Выпадающий список для выбора математической операции
- Кнопка "||" для выполнения расчета
- Поле результата с автоматическим обновлением
